## Overview
- 기간: 2022.06 ~ 2023.06 (약 14개월)
- 고객사: 원익 PNE
- 소속: 제이원소프트
- 역할: 개발 및 셋업
- 주요 업무: 공정 관리 기능 개발, 데이터 연동

## Background
2차전지 활성화/충방전 공정 전반을 통합 모니터링하고, 설비/PLC와 데이터 연동을 통해
생산 이력 및 공정 상태를 안정적으로 관리하는 IT 시스템을 구축했습니다.

## Key Features
- 라인 장비 상태 모니터링 대시보드(자동조립, Degas, Formation, Aging, EOL 등)
- 공정별 미들웨어 구성 및 설비 PC 위치에 따른 화면/기능 제어
- 트레이/셀 매핑 관리 및 이력 조회
- 레시피/Route 관리 화면 구성 및 공정 데이터 관리
- PLC 신호 기반 이벤트 처리 및 로그/이력 기록

## What I Built
- WinForms + DevExpress 기반 관리/모니터링 UI 구현
- PLC 통신 미들웨어 구조 설계 및 이벤트 핸들링
- EF Core 기반 데이터 모델 연동 및 공정 이력 저장
- 서버 API 연동 및 운영 환경 셋업 지원

## Stack
- C#, WinForms, DevExpress, MS-SQL, .NET Core, EF Core, REST API
